# PullTheRise
CraftRise kaynağından hashs.php dosyası sayesinde tüm dosyaları indirir


## API_URL:
```bash 
https://client.craftrise.network/api/launcher/hashs.php
```

## Kaynak:
> ⚠️ Dosya listesini almak için API'ye bakınız!
```bash 
https://client.craftrise.network/api/launcher/files/
```
## Manuel Curl ile dosya çekme komudu:
Örnek dosya: launcher.jar
### Cmd 
```bash 
curl https://client.craftrise.network/api/launcher/files/launcher.jar -o launcher.jar
```
### PS (PowerShell) 
```bash 
curl.exe https://client.craftrise.network/api/launcher/files/launcher.jar -o launcher.jar
```
